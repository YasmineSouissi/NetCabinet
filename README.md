# NetCabinet: Application Web pour la Gestion de Cabinet Médical

Bienvenue dans NetCabinet, une application web développée avec Symfony pour la gestion de cabinets médicaux. Cette application permet aux médecins de gérer leurs ordonnances et aux pharmaciens de gérer les médicaments et les ordonnances.

## Description du Projet

NetCabinet est une application web construite avec Symfony pour faciliter la gestion des cabinets médicaux. Elle offre des comptes séparés pour les médecins et les pharmaciens, permettant ainsi une gestion spécialisée des ordonnances et des médicaments.

## Fonctionnalités

- **Compte Médecin** : Gestion des ordonnances.
- **Compte Pharmacien** : Gestion des médicaments et des ordonnances.
- **Page d'Accueil** : Page de connexion pour accéder aux différentes sections de l'application.
- **Back-office Médecin** : Interface pour les médecins pour créer, modifier et supprimer des ordonnances.
- **Back-office Pharmacien** : Interface pour les pharmaciens pour gérer les médicaments et consulter les ordonnances.

## Prérequis

Avant de commencer, assurez-vous d'avoir les éléments suivants installés :

- [PHP](https://www.php.net/) (version 7.4 ou supérieure)
- [Composer](https://getcomposer.org/)
- [Symfony CLI](https://symfony.com/download)
- Un serveur de base de données (par exemple, MySQL)
